# ConfigFlow

ConfigFlow is a metadata-driven dashboard builder built using Next.js 14, Prisma, PostgreSQL, and NextAuth.

## Features

* Dynamic JSON-based UI rendering
* Drag and drop dashboard builder
* Authentication system
* PostgreSQL database integration
* Save and load configurations
* CSV import support
* Theme switching
* Dynamic component rendering
* Responsive dashboard layout

## Tech Stack

* Next.js 14
* TypeScript
* Prisma ORM
* PostgreSQL
* NextAuth
* Tailwind CSS

## Live Demo

Deployed on Vercel.

## Installation

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
